# Data Immersion & Wrangling Project

## Overview
This project focuses on understanding and preparing a dataset for analysis. 
The goal is to explore the data, identify potential issues, and clean the dataset so it can be used for further data analysis.
Due to large dataset size, First I created a sample dataset from a large dataset, then I performed data cleaning and feature engineering on the sample dataset.


## Objective
The main objective of this task is to learn the first step of any data analysis process, which is data cleaning and preprocessing.

## Tools Used
- Python
- Pandas
- Visual Studio Code

## Steps Performed

### 1. Data Access & Familiarization
First, the dataset was loaded using Python and Pandas.  
I explored the structure of the dataset to understand the different columns and types of data available.

### 2. Data Quality Assessment
The dataset was checked for common data issues such as:
- Missing values
- Duplicate records
- Incorrect formatting

### 3. Data Cleaning & Transformation
To improve the dataset quality:
- Duplicate rows were removed
- Problematic rows were skipped
- The dataset was processed to make it ready for analysis

### 4. Feature Engineering
Created a new column **Age_Group** based on age values.


## Files in This Repository
- Dataset.csv -> Original dataset
- sample_dataset.csv → Sample of original dataset
- cleaned_sample_dataset.csv → Cleaned dataset
- Data_cleaning.py → Python code used for cleaning

## Output
A cleaned dataset was generated and saved as **cleaned_sample_dataset.csv**.

## Conclusion
This task helped me understand how important data cleaning is before performing any data analysis or machine learning tasks.
