# Exploratory Data Analysis (EDA) & Business Intelligence

## Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on a retail sales dataset to extract meaningful insights and answer business-related questions.
Using Python, SQL, and data visualization techniques, the project identifies patterns, trends, and relationships in the dataset.

The results are summarized in a **dashboard mock-up** that highlights key business metrics.

---

## Objectives

* Understand the dataset structure and variables.
* Perform **descriptive statistics and univariate analysis**.
* Visualize the data using charts and graphs.
* Write **SQL queries** to answer business questions.
* Perform **multivariate analysis and correlation analysis**.
* Design a **static dashboard mock-up** to present key KPIs.

---

## Dataset

The dataset contains retail transaction data with the following columns:

* Invoice
* StockCode
* Description
* Quantity
* InvoiceDate
* Price
* Customer ID
* Country

This dataset represents product purchases made by customers from different countries.

## Exploratory Data Analysis

### 1. Data Loading

The dataset was loaded using the **Pandas library** to begin analysis.

### 2. Descriptive Statistics

Statistical summaries were generated to understand numerical variables such as:

* Mean
* Minimum
* Maximum
* Standard deviation

### 3. Univariate Analysis

Data distributions were visualized using:

* Histograms (Quantity and Price)
* Bar charts (Country distribution)

These visualizations help understand how data values are distributed.

---

## SQL Business Questions

Several business-related questions were analyzed using SQL queries.

Examples include:

1. What are the **top 5 products by total quantity sold**?
2. Which **country generates the highest sales**?
3. What is the **average product price**?
4. Which **customers spend the most money**?
5. What is the **total quantity sold by country**?

SQL queries help extract insights from the dataset for business decision-making.

---

## Multivariate Analysis

Relationships between variables were explored using:

* **Scatter Plot** (Quantity vs Price)
* **Pair Plot** (relationships between numerical variables)
* **Correlation Heatmap**

These techniques help identify relationships between sales variables.

---

## Dashboard Mock-up

A **static dashboard mock-up** was created in PowerPoint to display key business metrics.

Key Performance Indicators (KPIs) included:

* Total Sales
* Total Quantity Sold
* Total Customers
* Average Product Price

The dashboard also includes visual charts such as:

* Sales by Country
* Quantity vs Price Scatter Plot
* Correlation Heatmap

This dashboard provides a clear overview of important business insights.
