# Deep Dive Analysis and Interactive Dashboard

## Objective
The objective of this project is to analyze retail transaction data to identify key performance indicators and perform customer segmentation.

## KPIs
Total Revenue
Total Orders
Average Order Value
Total Customers

## Deep Dive Analysis
Customer segmentation was performed based on total customer spending.

Customers were divided into:
High Value Customers
Medium Value Customers
Low Value Customers

## Key Insights
High value customers generate the highest revenue.
United Kingdom contributes the majority of sales.
Customer segmentation helps businesses target marketing strategies.
